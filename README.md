# npm-package-starter

Repositories that can be used as a starter in creating npm package.

| dependencies | version |
| ------------ | ------- |
| pnpm         |  7.5.0  |
| typescript   |  4.8.4  |

## Usage

Click "**Use this template**" here or **clone** this repository.

### build
